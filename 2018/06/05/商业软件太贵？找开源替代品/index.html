商业软件太贵？找开源替代品

发表于 2018-06-05 | 分类于 files |

摘要：某些商业软件素以昂贵著称。随着云计算的日益普及，很多常用软件包供应商将一次性收费改为月租模式。虽然月租费貌似便宜，但长时间累积起来的租金还是很高的。这个时候我们就该考虑是否将目光转移到开源替代品上。

100个替代昂贵商业软件的开源应用面对大，中，小企业和家庭用户，立竿见影显著降低成本的开源软件。某些商业软件素以昂贵著称。随着云计算的日益普及，很多常用软件包供应商将一次性收费改为月租模式。虽然月租费貌似便宜，但也经不起长时间的累积。

尽管有许多好理由，但避免或减少使用费，仍然是许多用户看中开源应用软件的主要因素。基于这一点，我们更新了可替代昂贵闭源应用的开源应用软件年度清单。

今年的软件数量比以往任何时候都多， 入选的100个软件不失为大中小型企业和家庭用户的上佳选择。只要有可能，我们会提供相应付费软件的价格。请注意，虽然我们清单上的开源应用程序提供与其替代的闭源应用相对等的功能，但它们并不等同于彼此。

与往常一样，请使用每页下方的注释部分，提供你所了解的应该列入我们清单内的其它开源应用程序。

财会类软件

  1. Edoceo Imperium：专为中小型企业（SMB）开发的 Edoceo Imperium 将会计、发票和基本的客户关系管理（CRM）功能结合并集成了谷歌的日历，地图和电子邮件（Gmail）功能，支持多达四十亿个不同访问级别的用户。支持的操作系统：无限制

  2. FrontAccounting：同样为中小企业设计的 FrontAccounting 包括一些基本的企业资源规划（ERP）功能，如库存和生产跟踪。该公司网站提供帮助潜在用户体验其产品的演示。支持的操作系统：无限制

  3. TurboCASH：这一会计软件包旨在提供大多数中小企业所需的全部功能包括债务，债权，总帐，库存管理，发票， 调账，试算表和报告等。支持多公司和多用户。支持的操作系统：Windows

  4. XIWA：仅在 Linux 环境运行的 XIWA 意为 ” XIWA 是网络会计”。 自1999年开发至今，它具有支持多用户和多公司，工资，股票/投资跟踪，报表等功能。支持的操作系统：Linux

录音/编辑类软件

  1. Ardour：为满足专业人士的需要而开发，Ardour 提供全套录音，编辑和混音工具。主要特点包括无限量音轨和数据总线，支持破坏性/非破坏性编辑录音模式，支持数十种文件格式，“业内最灵活的混音器架构”，数以百计的插件，精确自动化采样，视频时间轴和更多的支持。支持的操作系统：Linux，OS X

  2. Audacity 是一个易于使用的录音和编辑应用程序，对于业余音乐人和家庭用户是一个很好的选择。它可将磁带，唱片和光盘转换成数字文件，并支持 WAV，AIFF，FLAC，MP2，MP3 或 Ogg Vorbis 文件格式。支持的操作系统：Windows，Linux 和 OS X

  3. Frinika 自称为一个“完整的音乐工作站”。它的用户界面和功能的先进性尚未达到一些专业人士的要求，但它提供音序控制，MIDI 接口支持, 软件合成，录音和乐谱编辑等功能。支持的操作系统：无限制

  4. Linux MultiMedia Studio (LMMS) 包括歌曲编辑，节拍和贝斯编辑，钢琴卷帘窗，64通道 FX 调音和更多功能。最初为 Linux 开发，现有 Windows 版本。支持的操作系统：Windows 和 Linux

  5. WaveShop：与前面几个功能齐全的软件相比，只在 Windows 环境下运行的 WaveShop 强调快速和小巧。值得关注的特点包括调音，淡化，采样率转换，音频生成，无限量撤销，广泛的帮助，峰值，方均根（RMS）和频谱分析。支持的操作系统：Windows

商业智能类软件

  1. JasperSoft：作为最流行的开源商业智能工具之一，JasperSoft 提供可视化企业信息仪表板，自助报表，丰富的分析，移动支持等更多功能。该公司还在 Jaspersoft.com 提供付费的商业版本。支持的操作系统：无限制

  2. Jedox：原名 Palo BI 的 Jedox 是一个完整的商业智能套件，包括一个联机分析处理(OLAP)服务器，Excel 插件，数据提取，转换和加载（ETL）和移动支持。基础版本只包括联机分析处理(OLAP)服务器和有限版 Excel 插件，可在开源许可证下免费使用。企业內部部署和云版本需付费。支持的操作系统：无限制

  3. Pentaho 提供数据集成，业务分析和大数据工具。开源社区版本可从 community.pentaho.com 下载。 用户包括汉莎航空，哈里伯顿，纳斯达克和江户国际等企业。 支持的操作系统：Windows，Linux 和 OS X

  4. Spago 声称是“唯一100％开源的，完整和灵活的商业智能套件。”它建立在其它开源项目的广泛基础上，提供模块化的报表，联机分析处理(OLAP)，图表，关键性能指标（KPI），互动管理驾驶舱，自助报表，位置智能，数据挖掘，信息仪表板，团队协作，数据提取，转换和加载（ETL），主数据管理等等。支持的操作系统：无限制

  5. KNIME：用户遍及60多个国家，KNIME 自称是“领先的数据驱动创新开放平台帮助企业引领变革。”其核心分析软件为开源免费，商业扩展则为有偿服务。支持的操作系统：Windows，Linux 和 OS X

  6. OpenI：这一插件扩展了 Pentaho 和 Jasper 的分析工具能力承诺让新用户在短短72小时内运行起来。它包括信息仪表板，交互式报表，数据提取，转换和加载（ETL），维度模型（dimensional modeling），多维联机分析处理(OLAP)设计和预测建模。支持的操作系统：无限制

  7. BIRT：功能范围略逊于清单上的其它软件，BIRT 专注于报表创建和可视化嵌入式应用。它的赞助公司为Actuate（提供各种相关的资源），Innovent Solutions 和 IBM。需要注意的是，你需要在 Eclipse 开发环境下使用它。支持的操作系统：Windows，Linux 和 OS X

  1. Dia：同 Visio 一样，Dia可以创建关系图，流程图，网络图，组织结构图和类似的图表。在默认情况下它支持 XML 文件，可以导出 EPS，SVG，XFIG，WMF 和 PNG 文件格式。支持的操作系统：Windows 和 Linux

  2. Gimp 是“GNU图像处理程序”的简称，它提供专业品质的照片编辑和图像制作，具有与 Photoshop 相同的高级功能，在 Linux 桌面用户中备受青睐。支持的操作系统：Windows 和 Linux

  3. Inkscape：像 Gimp 一样，Inkscape 适合专业平面设计师使用。它提供了创建矢量图形和支持多种不同文件类型的各种工具。支持的操作系统：Windows，Linux 和 OS X

数学类软件

  1. Sage 为超过100种不同的开源数学项目提供了一个统一的界面。它可以解析基础代数，微积分，数论，密码学，交换代数，群论，2D 和 3D 图形，线性图（linear graphics）等等。支持的操作系统：Windows，Linux 和 OS X

  2. Genius：像 Sage 一样，Genius 可处理多种类型数学运算，包括微积分，数论，统计，三角函数，模运算，2D 和 3D 绘图等等。它包括一个图形用户界面的开发环境（IDE），并可导出 LaTex、Troff 或 MathML 格式矩阵数据。支持的操作系统：Linux，OS X

  3. Scilab：许多教育机构使用 Scilab 辅助教授数学，科学和工程课程。它可以执行多种不同类型计算。Scilab 的企业版可提供有偿的支持，培训和服务。支持的操作系统：Windows，Linux 和 OS X

  4. Maxima 计算机代数系统可求解的数学表达式包括“微分，积分，泰勒级数，拉普拉斯变换，常微分方程，线性方程组，多项式，以及集合，数列，向量，矩阵和张量”等。它的代码基于 MIT 在20世纪60年代末开发的 MACSYMA 数学程序。支持的操作系统：Windows，Linux 和 OS X.

音乐类软件

  1. MuseScore：乐谱软件通常花费数百美元。但 MuseScore 创作优美的乐谱却是完全免费的。其特点包括无限和弦长度，无限制谱线数量，无限的声音输入数量，创作向导和模板，自动部分提取，换位，力度，歌词，和弦符号，打击乐谱符号，早期乐谱符号等等。支持的操作系统：Windows，Linux 和 OS X

办公效率

  1. OpenOffice：完整版 Microsoft Office 的价格近年来显著回落，但使用年费还是随着时间的推移而显著累积。相比之下，完整版本的 Apache OpenOffice 是完全免费的。它包括文字处理，电子表格，演示文稿，图片，数学公式编辑和数据库软件，它还可读写 Office 格式文件。支持的操作系统：Windows，Linux 和 OS X

  2. LibreOffice：从 OpenOffice 分支出来的 LibreOffice 提供其前身相同类型的软件并附带额外的功能扩展。其网站上的对比表有助于比较它与 Microsoft Office 的异同（注：在官方网站未找到此表。但在 Wiki 网可找到https://wiki.documentfoundation.org/Feature_Comparison:_LibreOffice_-_Microsoft_Office）。支持的操作系统：Windows，Linux 和 OS X

  3. Calligra 前身为 KOffice，Calligra 套件包括文字处理器，电子表格，演示文稿，项目管理，绘画（Krita），数据库（Kexi），矢量图形（Karbon），流程图和思维导图。它支持微软的文件格式，但用户界面与 Office 很不同。支持的操作系统：Windows，Linux 和 OS X

  4. NeoOffice：同 LibreOffice 一样，NeoOffice 也是从 OpenOffice 分支出来的。但 NeoOffice 仅用于 Mac 电脑。最新版本支持 Yosemite 系统的新 iCloud 驱动，可从 Mac App Store 下载。支持的操作系统：OS X，iOS

操作系统类软件

  1. Ubuntu：Linux 最流行的桌面发行版，Canonical 的 Ubuntu 提供云，服务器，台式机，手机，平板电脑和电视版本。Ubuntu 的 Advantage 计划提供有偿系统管理和支持服务。

  2. Red Hat Enterprise Linux：“业界领先的企业级 Linux 平台，”Red Hat 是一个商业运作的操作系统，根据该公司介绍，用户中超过90%是财富500强企业。Red Hat 还提供了各种相关的开发工具，中间件和云计算软件。

  3. Fedora：如果您想使用 Red Hat 的 Linux 版本却不想支付企业支持费，你该用它的免费社区版Fedora，这包括已安装在数以百万计系统上的超过一万个免费应用程序。

  4. SUSE Linux Enterprise：SUSE 系统运行在伦敦证券交易所，Office Depot，Teradata 和沃尔格林，全球用户超过13,000个。它有服务器，桌面和云版本，所有版本的支持均需要付费。

  5. openSUSE：免费且不需支持合同，自称“思想开放者的 Linux”的 openSUSE 是一个稳定的，易于使用的操作系统，内置有1,000多个不同的应用程序。

  6. Linux Mint：易于量身定制的 Linux Mint 给用户四种不同桌面界面选择。其中某些界面与旧版Windows 非常相似，使它成为替代旧电脑上 Windows XP 受欢迎的选择。

  7. Debian：拥有超过37,500个应用程序的 Debian 是一个功能非常全的 Linux 发行版。它在拥有丰富Linux 使用经验的用户群体中非常受欢迎。

工具类软件

  1. PDFCreator：此应用程序可从几乎任何类型的文件创建 PDF 文件。它还包括专业功能如压缩和调整图像大小，加密，密码保护，文档合并等等。支持的操作系统：Windows

  2. jPDF Tweak：基于 Java 的 jPDF 自称为“PDF文件的瑞士军刀”。它可以合并，分割，旋转，重新排序，加水印，加密，修复和签署 PDF 文件。支持的操作系统：无限制

  3. PDFedit：每周下载上千次，这个工具允许用户编辑自己的 PDF 文件。它可以编辑文本，删除页面，更改图形和更多。支持的操作系统：Windows 和 Linux

  4. PDF SAM 是“PDF分割和合并”的简称，PDF SAM 基础版可以提取页面，合并多个文件，重新排序等等。捐赠至少一个欧元，您可下载增强版，它增加了加密，解密和元数据编辑功能。支持的操作系统：Windows，Linux 和 OS X

销售点终端（POS）

  1. Openbravo POS：这个基于网络的销售点终端（POS）解决方案集成了 Openbravo 的企业资源规划（ERP）软件。移动版可以让零售人员帮助客户在店内任何地方结算。支持的操作系统：Windows，Linux 和 OS X

  2. uniCenta：与 Openbravo POS 基于相同源代码的 uniCenta 是一个多语种的销售点终端（POS）系统十分适合在多功能设施使用，如带零售店的餐厅。它已被下载超过13万次。支持的操作系统：Windows，Linux，OS X

  3. Floreant POS：仅适于餐馆，Floreant 已被用在超过25,000个场所，包括一些丹尼（Denny’s）连锁餐厅。它可处理订货，厨房打印，钱屉，外卖，税收，打折，订单分组等等。支持的操作系统：Linux

项目管理类软件

  1. GanttProject 允许用户创建任务和里程碑，组织任务，制定相关约束，创建基线和创建基于甘特图的计划评审技术图（PERT）。它的下载量每周超过20,000次，备受青睐。支持的操作系统：Windows，Linux 和 OS X

  2. dotProject：该项目管理工具包括日历，甘特图，报告，资源调度，文件共享，论坛等等。安装和定制等服务由项目开发人员提供。支持的操作系统：Windows 和 Linux

  3. Onepoint Project 的功能包括工作结构分解，项目管道，成本基线，计划/实际比较，里程碑趋势分析，资源利用，甘特/计划分解，任务管理，说明注释，文件管理，自定义字段等。它针对不同规模的组织有多种版本，包括免费开源的基础和社区版。支持的操作系统：Windows，Linux 和 OS X。
