基于GitHub和Hexo搭建博客

发表于 2018-04-22 | 分类于 study |

搞了一下午终于把博客给搭建好了，也顺便把博客的搭建过程记录下来。同时这也是第一篇正式博文哦。

准备工具

  1. 博客是基于GitHub和Hexo，那就少不了安装点什么东西了。hexo基于Node.js的，官网下载,傻瓜化安装一路next就可以了。通过 node -v 来测试是否安装成功，安装成功会显示版本号。
  2. 接下来是Git下载地址直接到官网下载相应的版本，进行安装即可。安装完成后通过 git -version来测试安装是否成功。
  3. 接下来就是注册Git 账号了, 直接去Git 官网注册即可。注册完成后记得添加SSh Key，ssh key是用于认证，让github绑定到你这台机器。

配置GitHub Page

这个挺简单的，添加一个仓库New repository，但要注意的是仓库名一定是你的GitHub户名.github.io格式

启用GitHub Page

点击右边的“Setting”菜单进入设置,点击”Launch automatic page generator”
进入之后点击底部的”Continue to layouts”
之后随意选择模版，点击”Publish page”,发布github默认生成的一个静态站点
打开自己在github的静态网址，我的http://dontstayup.github.io
